# 基于多算法融合的可持续发展目标（SDG）系统建模与因果分析 2025.6 -2025.8 

开发工程师 

针对德国地方SDG指标的复杂交互关系，开发多算法融合建模框架，量化政治，经济变量对环境绩效（如电动车普及率、碳排放强度）的动态影响。 

• 数据挖掘与模式识别： 利用 Python 与 SQL 清洗整合 100+ 维异构指标，通过 VIF 检验与 Spearman 相关性分析进⾏特征筛选；引入 Louvain 社区发现算法构建指标关联⽹络，实现高维变量的逻辑解耦并识别出 5 类核⼼目标群。 

• 因果推断与效应评估： 结合 K-Means++ 与孤立森林（Isolation Forest） 实现城市画像分群与异常区域识别；构建 PSM-DID（ 模型评估政策对环境绩效的净影响，并利用因果森林（Causal Forest） 量化教育、收入等变量在不同分群城市中的因果效应异质性。 

成果：建立端到端分析框架，实现指标⽹络可视化与因果路径追踪，模型解释⼒提升约12%。产出定量分析报告，为不同产业背景的城市提供了针对性的政策优化路径。研究成果入选专业科研展示项目。
