# 基于 A/B 实验设计的广告投放增量归因与用户行为挖掘项目 2025.8 – 2025.10 

为解决广告归因中“自然转化”干扰的痛点，我利用 58 万条真实数据构建实验模型，旨在量化广告投放的纯增量价值（Incremental Lift）。 

• 计了从“统计预检”到“决策裁决”的完整闭环。首先通过 Levene's Test 发现方差齐性不符，果断采用 Welch's t-test 修正统计偏见，挖掘出转化人群存在显著的“频次阈值”（均值 84 次 vs 23 次，Cohen's d=1.42）。 

• 针对样本量 24:1 的极端失衡，运用 Power Analysis 严谨估算最小样本量并进⾏等比随机抽样，排除规模偏见。随后结合卡方检验与残差分析，构建多维交叉矩阵，精准锁定了周⼀及下午的高转化时间窗口。 

结果：实验确证广告将转化胜算（Odds）显著提升 41% 。基于数据挖掘结论，我为业务方提供了“确保高频触达”与“分时段差异化出价”的量化策略建议，实现了从统计发现到业务驱动的深度赋能。
