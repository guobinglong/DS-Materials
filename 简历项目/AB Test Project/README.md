# 广告转化增效量化归因分析专项基于AB检验 2025.8 – 2025.10 

项目背景：投放营销广告时面临转化贡献度不明的业务痛点， 无法确定用户转化是广告驱动还是自然购买， 且实验数据存在严重的样本不平衡， 需建立科学的增量归因体系以优化预算分配效率。

用戶行为特征挖掘：通过EDA定位转化关键驱动因子 ， 识别 “ 时间窗口”  与  “ 曝光频次”  为转化关键因子；利用IQR法清洗异常流量干扰， 运用卡方检验与T检验验证特征显著性， 发现 “ 周一 ”  时段及高频曝光下的高转化潜力 ， 为策略制定提供数据支撑。

实验设计：针对样本规模差异带来的统计偏差， 预先进行统计功效分析， 在α =0.05、 Power =0.8的约束下计算最小样本量， 对实验组执行等量随机抽样， 确保实验结果的可信度与统计显著性。

策略落地 ：综合运用卡方检验验证转化与广告曝光的关联性， 通过独立样本t检验定位最佳触达频次阈值， 最终以比例Z检验量化广告的净提升效应。结果证实广告驱动转化率从对照组的1.79%显著提升至2.55%， 计算得优势比为1.43， 即看广告用户的购买发生比是不看用户的1.43倍。基于此， 推动将预算向 “ 周一黄金时段”  聚合并确立  “ 高频次覆盖”  的触达标准， 有效提升广告预算的边际回报。
