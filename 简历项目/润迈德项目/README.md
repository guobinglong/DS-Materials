# 急性心肌梗死（STEMI）患者预后风险预测与 caIMR 指标建模分析 

针对临床上急性⼼梗患者术后微循环障碍难以无创评估的痛点，利用 541 例脱敏临床数据，旨在通过R 语言构建预后模型，量化评估 caIMR 指标对术后⼀年主要不良⼼血管事件（MACE）的预测价值，驱动临床分级管理。 

• 探索性分析与特征筛选： 负责 541 例样本的基线资料清洗。通过 t 检验及卡方检验对比 MACE 组与非 MACE 组，识别出 hs-cTnT（1143.0 vs 535.5 ng/L）、 LVEF 等 12 项具有统计学差异的显著特征（P<0.05），确定入模变量。 

• 独立风险因子建模： 构建单因素及多因素 Cox 比例风险回归模型。在控制年龄、⼼率及病变血管位置等混杂因素后，证实 caIMR 是预后的独立危险因素，其风险比 HR 值为 2.164（95% CI: 1.258~3.722, P<0.01）。 

• 模型效能评估与阈值挖掘： 运用 ROC 分析量化模型识别能⼒，计算 AUC 值为 0.688，并挖掘出预测 MACE 的最佳截断值为 25.3 U。 

结果：成功产出了以 calMR > 25.3 为核⼼的风险分层指标体系。验证发现 calMR>25.3 组的 MACE 发⽣率（21.8%）显著高于低值组（6.5%），且全因死亡风险大幅增加（Log-rank P<0.001）。成功实现了对术后高危患者的精准识别与分流管理。
